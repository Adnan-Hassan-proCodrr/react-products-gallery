# React Product Gallery
A React SPA that fetches products from an API and displays a dynamic product gallery with detail pages. Fully responsive with modern UI.
## Features
- Fetches products from FakeStore API using Axios
- Dynamic routing with React Router
- Product detail pages
- Responsive design for mobile & desktop
- Modern and interactive UI with hover effects
## Technologies Used
- React
- React Router DOM
- Axios
- CSS (custom styling)
- FakeStore API
